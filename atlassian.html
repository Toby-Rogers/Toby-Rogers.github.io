Context

Internal platform owner for the full Atlassian stack at InvestCloud (Cloud and Data Center) — sole administrator across Atlassian Jira, JSM, Confluence, and Tempo, serving all business units from Product and R&D through to Professional Services and Support. Work spans the full ALM lifecycle — requirements, design, implementation, validation, documentation, and training — across delivery methodologies (Waterfall, Agile, Lean, Scrum, Kanban, SAFe), DevOps and QA.

Key Outcomes
Platform consistency

Standardised Atlassian Jira instances across all business units supporting delivery and portfolio-level reporting.

Governance & auditability

Auditable processes around access, config changes, and usage meeting security and compliance requirements.

Actionable BI

Leadership reporting on productivity, utilisation, and initiatives built by connecting Atlassian Jira and Tempo into Microsoft Power BI.

Adoption & enablement

Higher tool adoption through hands-on training, process design, and ongoing consulting with teams.

Cloud migration & cost

Data Center → Cloud migrations cutting annual platform cost from ~$30k–$37k to ~$3k while preserving history, attachments, and comments.

Atlassian Jira Data Center → Cloud Migration
Jira Server / Data Center → Cloud migration (PB/APAC decommissioning)
Atlassian Jira · Cloud Migration · JCMA
  • Led Jira Server / Data Center to Jira Cloud migrations as part of the PB/APAC Jira decommissioning, moving client and internal projects into InvestCloud Cloud environments while preserving issue history, attachments, screenshots, and comments.
  • Resolved Atlassian Migration Assistant issues during project moves from PB Jira Data Center / Server to IC Jira Cloud, including domain trust and migration readiness problems.
  • Validated post-migration data integrity — issue key remapping, retained issue IDs, child issue behaviour, and downstream impacts on filters, release notes, and reporting.
Dedicated Atlassian Jira Cloud instance — MUFG client
Atlassian Jira Cloud · Atlassian Access · SSO/MFA
  • Designed and delivered a dedicated Jira Cloud instance for the MUFG client — project migration, client/internal project split, licensing/security model, and SSO/MFA design via Atlassian Access.
  • Built the migration business case, reducing annual platform cost from roughly $30k–$37k on Data Center to around $3k on Cloud, while improving security and maintaining a familiar client support model.
Impact

~90% reduction in annual platform cost with an improved security posture and no disruption to the client support model.
